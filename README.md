# Express_ejs_Template
## How to use
* クローン
  * `git clone https://github.com/nobkovskii/Express_ejs_Template.git`
* もろもろインストール
  * `npm install express ejs request axios path express-session --save`
  * `npm install -g express-generator`
* アプリ起動
  * `node app.js`
* ブラウザアクセス
  * http://localhost:3000/
